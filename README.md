# DataShadow

DataShadow is a premium privacy-defense browser extension designed to expose the "shadow data" websites collect.

## Project Structure
- `public/`: Static assets (icons, manifest, etc.)
- `src/`: Source code
  - `pages/`: UI HTML and Logic
  - `scripts/`: Background and Content scripts
- `index.html`: Entry point (Value Dashboard)

## Getting Started
1. Run `npm install`
2. Run `npm run dev` to start the development server.
3. To load in Chrome, use the `public/` directory (for dev) or `dist/` (after build).
