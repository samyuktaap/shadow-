FAQ

Got Questions?

Is DataShadow completely free? +
Yes — DataShadow is 100% free and open source. No subscriptions, no hidden fees, no premium tier. Ever.
Does DataShadow send my data to any server? +
Never. All analysis happens locally inside your browser. We have zero access to your browsing data — it never leaves your device.
Will it slow down my browser? +
No. DataShadow is built for performance. It runs as a lightweight background process with zero impact on page load speed.
Does it work on all websites? +
Yes. DataShadow works on every website you visit automatically — no configuration needed.
Do I need to create an account? +
No account, no sign-up, no email required. Install and it works instantly.
Is it safe to install extensions in Developer Mode? +
Yes — Developer Mode just means the extension isn't on the Chrome Web Store yet. The code is fully open source and auditable on our GitHub.
What browsers does DataShadow support? +
Currently Chrome, Brave, and Edge — any Chromium-based browser supports it.
